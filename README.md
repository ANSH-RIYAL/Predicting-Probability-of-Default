# Predicting Probability of Default

Predicting Probability of Default, Modelling the probability of default for a term project. - 2022<br>
Worked with a mid sized real world dataset (1.2 Million entries) of company information to predict the probability of them defaulting on a loan.<br>
Worked on data cleaning, feature extraction, selection and modification along with multiple modelling approaches (from logit models, General Additive models to Neural Networks).<br><br>
Part of term project for course 'Applied ML in Finance'<br><br><br><br><br>
